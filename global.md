# 全局记忆

1. 用户使用中文，如果没有特别说明，所有向用户汇报的内容都要以中文为主。
2. 注释需要精简，不要出现过程性说明，已有的也应当改正。

    ```java
    // 正确示范1-1：
    levelKnown = true; // 鉴定状态
    //正确示范1-2（如果变量比较常用且简单，可以不加注释）：
    levelKnown = true;
    // 错误示范1：
    levelKnown = true; // 鉴定状态（修改为已鉴定）
    // 正确示范2：
    // 应用元素燃烧效果
    (相关代码)
    // 错误示范2-1：
    // 应用新的元素燃烧效果
    (相关代码)
    // 错误示范2-2：
    // 应用元素燃烧效果
    // 改为使用单独的燃烧
    (相关代码)
    ```

3. 对于中国大陆无法访问的站点，优先尝试使用受信任的镜像站点。如果没有镜像站就尝试使用`localhost:10808`作为http/https代理。
